Detailed Description

RocketMotor Base Class This class is a pure virtual class that represents any kind of primary propulsion component intended to apply significant impulse to the rocket. Examples of this are solid/liquid/hybrid motors, or anything else that's designed to actually make the rocket liftoff and fly.

The class is a pure virtual class, meaning it only exists to define an interface that all rocket motor implementations should abide by.
