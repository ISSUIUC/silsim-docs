Detailed Description

ThrustCurveSolidMotor Derived Class Models a solid rocket motor that produces a varying magnitude of thrust as it burns. The thrust-vs-time relationship is provided in a thrust curve .csv file, which is then parsed into an Eigen matrix for future lookups.

The thrust getter functions perform simple linear interpolation based on the current simulation timestamp to lookup how much thrust the motor should be generating.
