◆ get_wind_vector()

Vector3d Atmosphere::get_wind_vector ( double  tStamp)

Generates a wind vector in the ENU frame with a direction and magnitude determined by the wind model used.

Wind model generates a random variance in both wind direction and magnitude at a determined rate. It then applies a low-pass filter to the variance so that the wind vector doesn't change abrupty and rather ebbs-and-flows much like real wind.

If wind variance is not enabled, the the return wind vector is simply the nominal wind

Parameters
tStampCurrent simulation timestamp
Returns
Vector3d Instantaneous wind vector
